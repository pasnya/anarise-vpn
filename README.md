# ANARISE VPN 🚀
## Бесплатный впн для андроид на базе NaiveProxy, Vless и Hysteria2

> *"Я не программист, но свободный интернет сам себя не настроит!"*

Добро пожаловать в **ANARISE** — простой, надежный и крайне упрямый **бесплатный впн для андроид**, ориентированный на удобный и безопасный обход блокировок. Проект представляет собой оптимизированный **naiveproxy android клиент** (а также **naiveproxy андроид клиент** для русскоязычных пользователей), собранный исключительно из качественных открытых библиотек и передовых сетевых решений.

### Почему именно ANARISE?
Найти рабочий, современный и стабильный **клиент NaiveProxy на Android** сегодня — огромная редкость. Большинство существующих мобильных решений либо заброшены разработчиками, либо требуют детального понимания сетевых технологий для ручной настройки конфигураций.

**ANARISE** решает эту проблему. Это полноценный, быстрый и функциональный **naiveproxy android клиент**, разработанный для людей. Он предоставляет готовый **бесплатный впн для андроид** прямо «из коробки» с удобным графическим интерфейсом на базе Jetpack Compose, автоматическим импортом конфигураций, проверкой пинга и гибкой маршрутизацией.

## Что умеет ANARISE?

Приложение работает на базе мощного ядра **Xray-core** (через легковесный туннель) и поддерживает передовые протоколы для сохранения свободного доступа к сети:

*   **NaiveProxy (Главная фича!)** — надежный обход блокировок. Наш **naiveproxy андроид клиент** работает поверх протокола HTTP/2 и идеально маскируется под обычный веб-трафик браузера Chrome. Блокировщики видят стандартную сессию веб-серфинга, что делает этот протокол лучшим выбором для обхода сложных DPI-систем.
*   **Внешние списки серверов (Новое!)** — встроенная вкладка «Внешние» позволяет загружать и автоматически тестировать публичные конфигурации из популярных репозиториев. Приложение проверяет доступность серверов через TCP-пинг и выводит только реально работающие серверы, отсортированные по минимальной задержке.
*   **Умный DNS-обход через NaiveProxy**: Настроен автоматический резолвинг DNS через TCP и безопасный обход ограничений UDP, что гарантирует стабильное соединение без утечек.
*   **VLESS** — классический высокоскоростной протокол для повседневного использования.
*   **VMess** — проверенная временем технология для стабильного туннелирования трафика.
*   **Hysteria2** — сверхбыстрый протокол на базе QUIC (UDP). Обеспечивает максимальную скорость и низкую задержку. Поддерживает обфускацию **Salamander** для маскировки UDP-трафика.
*   **Раздельное туннелирование (Split Tunneling)**: Вы можете выбрать, какие приложения пускать через VPN, а какие (например, мобильные банки или государственные порталы) оставить работать напрямую.
*   **Мгновенный пинг**: Проверка задержки до всех добавленных серверов в один клик.

## Архитектура проекта (Под капотом)

- **Jetpack Compose & Material 3** — стильный и современный темный интерфейс.
- **Модульная структура** — экраны настроек, туннелирования и дашборда разнесены по независимым пакетам. Никакого спагетти-кода.
- **vyom-tun-sdk** — собственное SDK для низкоуровневого взаимодействия с туннелем на C++ и Go (Xray).
- **Hysteria2** — официальный бинарник Hysteria2 для Android (arm64), запускаемый как отдельный процесс.

## Как собрать APK самостоятельно?

Если вы хотите собрать этот **бесплатный впн для андроид** своими руками, вам понадобятся установленный Android SDK и Java 17.

Запустите в терминале:
```bash
./gradlew assembleDebug
```
Скомпилированный файл APK появится по пути: `app/build/outputs/apk/debug/app-debug.apk`.
