# ANARISE VPN 🚀 — Бесплатный VPN для Андроид и Windows (NaiveProxy, Hysteria2, VLESS)

## VPN-клиент со втроенными публичными конфигурациями

**ANARISE VPN** — это легкий, быстрый и полностью бесплатный VPN-клиент с открытым исходным кодом для Android и Windows. Приложение разработано для надежного обхода блокировок (DPI) и защиты приватности в сети с помощью передовых протоколов маскировки трафика.

## 📱 Клиент Android (`/android`)

Полноценный и быстрый **бесплатный vpn для андроид** с удобным графическим интерфейсом. Это надежный **naiveproxy android client** и **vless android клиент**, поддерживающий обход современных систем DPI.

### Основные фичи
*   **Поддержка NaiveProxy** — обход блокировок с маскировкой под стандартный трафик Chrome (HTTP/2).
*   **Hysteria2 & VLESS** — высокая скорость, низкая задержка и поддержка обфускации **Salamander**.
*   **Вкладка «Внешние»** — автоматическая загрузка, параллельная проверка пинга и фильтрация публичных серверов.
*   **Раздельное туннелирование (Split Tunneling)** — выбор приложений, работающих через VPN.
*   **Быстрый пинг** всей истории в один клик.

### Сборка
Требуются Android SDK и Java 17.
```bash
cd android
./gradlew assembleRelease
```
Скомпилированный APK появится по пути: `app/build/outputs/apk/release/app-release.apk`.

## 💻 Клиент Windows (`/windows`)

Легковесный десктопный **vpn клиент windows** на базе .NET 8 WPF, использующий WebView2 для отрисовки потрясающего интерфейса.

### Основные фичи
*   **Системный прокси-режим** — автоматическое переключение системных настроек прокси Windows (SOCKS5/HTTP на localhost) при подключении. Не требует прав администратора.
*   **Автозагрузка ядер** — при первом запуске приложение само скачивает стабильные бинарные сборки `xray.exe` и `hysteria.exe` с официальных релизов на GitHub.
*   **Интерактивный лог** — вывод логов работы ядер в реальном времени прямо на вкладке логов.
*   **Автозапуск** и сворачивание в системный трей.
*   **Внешние списки** с автоматической фильтрацией и сортировкой по TCP-пингу.

### Сборка
Требуется .NET 8.0 SDK.
```powershell
cd windows
dotnet publish -c Release -r win-x64 --self-contained false
```
Результат сборки будет находиться в папке `bin/Release/net8.0-windows/publish/`.
