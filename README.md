# ANARISE VPN 🚀

> *"Я не программист, но свободный интернет сам себя не настроит!"*

Добро пожаловать в **ANARISE** — простой, надежный и крайне упрямый Android-клиент для обхода блокировок. 

## Предыстория (Как мы до этого докатились)
Автор этого проекта — **не программист** (и, честно говоря, гордится этим). Но в мире, где блокировки VPN-протоколов сыплются чаще, чем обновления Windows, пришлось брать дело в свои руки. 

Когда стандартные протоколы один за другим начали превращаться в "тыкву", мы сели, подумали и довели это приложение до ума. Теперь это не просто инструмент, а настоящий выживальщик в мире цифровых стен.

---

## Что умеет ANARISE?

Приложение работает на базе мощного ядра **Xray-core** (через легковесный туннель) и поддерживает всё самое современное для сохранения доступа к знаниям, мемам и работе:

*   **VLESS** — классика жанра. Быстро, надежно, незаметно.
*   **VMess** — проверенный временем протокол для обхода стандартных ограничений.
*   **NaiveProxy (NEW!)** — наша тяжелая артиллерия. Работает поверх HTTP/2 и маскируется под обычный веб-трафик браузера Chrome. Блокировщики думают, что вы просто читаете Википедию, а на самом деле вы... ну, сами знаете.
*   **Умный обход DNS-блокировок**: Специально для NaiveProxy мы настроили автоматический резолвинг DNS через TCP и безопасный обход UDP-ограничений.
*   **Раздельное туннелирование (Split Tunneling)**: Вы можете выбрать, какие приложения пускать через VPN, а какие (например, банковские клиенты или госуслуги) оставить работать напрямую.
*   **Быстрый пинг**: Проверка задержки до всех ваших серверов в один клик.

## Архитектура проекта (Под капотом)

Проект собран максимально аккуратно (мы провели масштабный рефакторинг и избавились от 1000+ строк "лапшичного" кода):
- **Jetpack Compose & Material 3** — стильный и современный темный интерфейс.
- **Модульная структура** — экраны настроек, туннелирования и дашборда разнесены по независимым пакетам. Никакого спагетти-кода.
- **vyom-tun-sdk** — собственное SDK для низкоуровневого взаимодействия с туннелем на C++ и Go (Xray).

## Как собрать APK самостоятельно?

Если вы хотите собрать приложение своими руками, вам понадобятся установленный Android SDK и Java 17.

Запустите в терминале:
```bash
./gradlew assembleDebug
```
Скомпилированный файл APK появится по пути: `app/build/outputs/apk/debug/app-debug.apk`.
