# ANARISE VPN 🚀
## Легковесные бесплатные VPN-клиенты для Android и Windows на базе Hysteria2, NaiveProxy и VLESS

Добро пожаловать в репозиторий **ANARISE VPN** — кроссплатформенного проекта, созданного для быстрого, удобного и надежного обхода интернет-блокировок. Проект содержит два полноценных клиентских приложения, разделяющих единый визуальный стиль (стильный Cyberpunk/Dark-Yellow дизайн с акцентным желтым цветом `#FFCC00`) и ориентированных на максимальную простоту использования «из коробки».

---

## Структура репозитория

Репозиторий разделен на две основные части:

*   📂 [/android](file:///d:/Сайты/Anarise/android) — Мобильный клиент под Android (Kotlin, Jetpack Compose, Material 3, vyom-tun-sdk на Go/C++, ядро Hysteria2).
*   📂 [/windows](file:///d:/Сайты/Anarise/windows) — Десктопный клиент под Windows (.NET 8.0 WPF, WebView2 HTML/CSS/JS frontend, управление ядрами Xray-core и Hysteria2).

## 📱 Клиент Android (`/android`)

Полноценный и быстрый **NaiveProxy & Hysteria2 android-клиент** с графическим интерфейсом.

### Основные фичи
*   **Поддержка NaiveProxy** — обход блокировок с маскировкой под стандартный трафик Chrome (HTTP/2).
*   **Hysteria2 & VLESS** — высокая скорость, поддержка обфускации **Salamander**.
*   **Вкладка «Внешние»** — автоматическая загрузка, параллельная проверка пинга и фильтрация публичных серверов.
*   **Раздельное туннелирование (Split Tunneling)** — выбор приложений, работающих через VPN.
*   **Быстрый пинг** всей истории в один клик.

### Сборка
Требуются Android SDK и Java 17.
```bash
cd android
./gradlew assembleDebug
```
Скомпилированный APK появится по пути: `app/build/outputs/apk/debug/app-debug.apk`.

## 💻 Клиент Windows (`/windows`)

Легковесный десктопный клиент на базе .NET 8 WPF, использующий WebView2 для отрисовки потрясающего интерфейса.

### Основные фичи
*   **Системный прокси-режим** — автоматическое переключение системных настроек прокси Windows (SOCKS5/HTTP на localhost) при подключении. Не требует прав администратора и виртуальных сетевых карт.
*   **Автозагрузка ядер** — при первом запуске приложение само скачивает стабильные бинарные сборки `xray.exe` и `hysteria.exe` с официальных релизов на GitHub.
*   **Интерактивный лог** — вывод логов работы ядер в реальном времени прямо на вкладке логов.
*   **Автозапуск** и сворачивание в системный трей.
*   **Внешние списки** с автоматической фильтрацией и сортировкой по TCP-пингу.

### Сборка
Требуется .NET 8.0 SDK.
```powershell
cd windows
dotnet publish -c Release -r win-x64 --self-contained false
```
Результат сборки будет находиться в папке `bin/Release/net8.0-windows/publish/`.
